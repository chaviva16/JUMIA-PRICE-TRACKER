A simple **Streamlit** app to search for products on **Jumia Nigeria** and see their **name, price, image, and link**.  

**Live demo:** [https://jumia-price-tracker.streamlit.app/](https://jumia-price-tracker.streamlit.app/)

---

##  Features

- Search by product name or pick from popular products.  
- Displays product image, title, price, and link.

## How to Run

1. Clone the repo:  
   ```bash
   git clone https://github.com/chaviva16/JUMIA-PRICE-TRACKER.git
   cd JUMIA-PRICE-TRACKER
   Install dependencies:

2.Install dependencies:

pip install streamlit requests beautifulsoup4

3.Add ScraperAPI key in .streamlit/secrets.toml:

SCRAPER_API_KEY = "your_api_key_here"

4. Run the app:

streamlit run app.py
